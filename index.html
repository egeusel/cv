Highlighted Projects

Gitlet

  • Coded Gitlet project, a command line program and a version control system using Java; the project architecture is inspired by the Git workflow, and has commands such as init, branch, checkout, and log, analogous to the original Git versions
  • When the init command is used, a hidden .gitlet repo is created in the current working director; it keeps track of the files that are staged to be added, staged to be removed as well as those that are deleted, edited, or newly created (untracked) since the last commit
  • Previous versions of files in the directory are saved using the Sha-1 code, the unique identifier of a file, and ensures memory efficiency of VCS; file opera- tions are handled using the java.nio and java.io packages and the last state of the program is preserved using the Java Serialization API

INTERNET PROTOCOL IMPLEMENTATION

  • Implemented Distance Vector Protocol in Python for intradomain routing which computes efficient paths across the network; for this, a distributed algorithm running at each router was used; split horizon, poison reverse, and route poisoning were utilized to ensure time efficiency and to avoid congestion
